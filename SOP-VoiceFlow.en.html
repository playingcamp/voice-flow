SOP — VoiceFlow

🌍 English · Español  |  Author: Carlos Alcalá  ·  Version: 1.0  ·  Date: May 10, 2026  ·  Stack: Whisper local + Gemini 2.5 Flash + Python + AppleScript
What VoiceFlow is. A local, agentic alternative to Wispr Flow. Push-to-talk for macOS that (1) dictates with AI formatting, (2) executes actions (email, WhatsApp, calendar, etc.) and (3) replaces selected text with voice transformations. Built in one afternoon with Claude Code. Cost: ~$0/month (Gemini free tier covers personal use).

1. Architecture

┌─────────────────────────────────────────────────────────────┐
│  Right Option key (push-to-talk)                            │
│         ↓                                                    │
│  sounddevice records audio (while held)                     │
│         ↓ on release                                         │
│  Local Whisper "base"  ──> transcribe (ES/EN/mix auto)      │
│         ↓ text                                               │
│  ┌─ Exact snippet? ──> paste snippet, END                   │
│  └─ NO                                                       │
│         ↓                                                    │
│  Gemini 2.5 Flash with function calling (8 tools)           │
│         ↓                                                    │
│  ┌─ paste_text         → Cmd+V at cursor                    │
│  ├─ open_app/close_app → `open -a` / AppleScript quit       │
│  ├─ search_web         → Chrome + google.com/search         │
│  ├─ compose_email      → Gmail Web with draft               │
│  ├─ send_whatsapp      → whatsapp:// URL scheme             │
│  ├─ create_calendar    → Google Calendar URL                │
│  └─ create_reminder    → AppleScript to Reminders.app       │
└─────────────────────────────────────────────────────────────┘
REPLACE mode: ⌥ + ⌘ → Cmd+C → selection + instruction → Gemini → Cmd+V

Required macOS permissions

Permissions are granted to VoiceFlow.app, not to python3. The bundle ID com.voiceflow.local is signed ad-hoc at build time so that TCC identifies it consistently across rebuilds.

4. Features

AI-enhanced dictation ✓

  • Auto multi-language: ES, EN, mixed (Whisper detects)
  • Auto-punctuation, accents, capitalization via Gemini
  • Remove fillers: strips "uh", "um", "like", "you know"
  • Backtrack: "Meet at 2... actually 3" → "Meet at 3"
  • Lists: detects explicit markers ("point A", "first, second") and formats them
  • Personal vocabulary: respects capitalization/spelling of terms in dictionary.json

Local snippets ✓

Exact match (case-insensitive, no punctuation). Skips Gemini → ~0 ms latency.

Agentic actions ✓

See table in section 5.

Replace selected text ✓

Select text → ⌥ + ⌘ → speak instruction → replaces.

6. Customization

dictionary.json — personal vocabulary

Edit and restart the app:

{
  "vocabulary": ["YourBrand", "ProperNoun", "TechnicalJargon"],
  "context": "Brief description of you and your work..."
}

Effects:

  • Injected into Whisper's initial_prompt → improves transcription of those terms
  • Injected into Gemini's system instruction → enforces correct spelling
  • The context guides how Gemini picks tools (knows what you mean by your jargon)

snippets.json — voice shortcuts

{
  "snippets": {
    "email signature": "Best,\\nYour Name",
    "my intro": "Hi, I'm Y and I do X..."
  }
}

If you dictate exactly the key (no trailing punctuation, case-insensitive) → the value is pasted without going through Gemini.

8. Troubleshooting

SymptomProbable causeSolution
This process is not trusted! in err log Missing Accessibility / Input Monitoring permission for VoiceFlow System Settings → Privacy → enable VoiceFlow in both
⌥ key not detected Permissions not applied to current binary tccutil reset Accessibility com.voiceflow.local and re-grant
"No tool call" or weird Gemini response Quota exhausted or connectivity Check tail ~/voice-flow/voiceflow.err.log
Bad transcription base model too small Change MODEL_NAME = "small" in voiceflow.py (more RAM and CPU)
Replace mode doesn't capture selection App doesn't respond to standard Cmd+C Manually verify that Cmd+C copies in that app
TCC limitation. If you rebuild VoiceFlow.app or change its bundle ID, permissions are invalidated. You'll need to re-add it manually or run tccutil reset Accessibility com.voiceflow.local.
