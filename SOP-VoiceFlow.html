SOP — VoiceFlow

🌍 English · Español  |  Autor: Carlos Alcalá  ·  Versión: 1.0  ·  Fecha: 10 mayo 2026  ·  Stack: Whisper local + Gemini 2.5 Flash + Python + AppleScript
Qué es VoiceFlow. Alternativa local y agentic a Wispr Flow. Push-to-talk para macOS que (1) dicta con formato AI, (2) ejecuta acciones (email, WhatsApp, calendar, etc.) y (3) reemplaza texto seleccionado con transformaciones de voz. Construido en una tarde con Claude Code. Coste: ~0€/mes (free tier de Gemini cubre uso personal).

1. Arquitectura

┌─────────────────────────────────────────────────────────────┐
│  Tecla ⌥ derecha (push-to-talk)                             │
│         ↓                                                    │
│  sounddevice graba audio (mientras se mantiene pulsada)     │
│         ↓ al soltar                                          │
│  Whisper local "base"  ──> transcribe (ES/EN/mezcla auto)   │
│         ↓ texto                                              │
│  ┌─ Snippet exacto? ──> pega snippet, FIN                   │
│  └─ NO                                                       │
│         ↓                                                    │
│  Gemini 2.5 Flash con function calling (8 tools)            │
│         ↓                                                    │
│  ┌─ paste_text         → Cmd+V en cursor                    │
│  ├─ open_app/close_app → `open -a` / AppleScript quit       │
│  ├─ search_web         → Chrome + google.com/search         │
│  ├─ compose_email      → Gmail Web con borrador             │
│  ├─ send_whatsapp      → whatsapp:// URL scheme             │
│  ├─ create_calendar    → Google Calendar URL                │
│  └─ create_reminder    → AppleScript a Reminders.app        │
└─────────────────────────────────────────────────────────────┘

4. Features

Dictado AI-enhanced ✓

  • Multi-idioma auto: ES, EN, mezcla (Whisper detecta)
  • Auto-puntuación, acentos, mayúsculas vía Gemini
  • Remove fillers: quita "eh", "este", "o sea", "um", "uh"
  • Backtrack: "Quedamos a las 2... mejor a las 3" → "Quedamos a las 3"
  • Listas: detecta marcadores explícitos ("punto A", "primero, segundo") y formatea
  • Vocabulario personal: respeta mayúsculas/ortografía de términos en dictionary.json

Snippets locales ✓

Match exacto (case-insensitive, sin puntuación). Salta Gemini → latencia ~0ms.

Acciones agentic ✓

Ver tabla en sección 5.

Replace selected text ✓

Selecciona texto → ⌥ + ⌘ → habla instrucción → reemplaza.

6. Personalización

dictionary.json — vocabulario personal

Editar y reiniciar la app:

{
  "vocabulary": ["Playing Camp", "Kanbox", "Apify", ...],
  "context": "El usuario es Carlos, fundador de Playing Camp..."
}

Efectos:

  • Se inyecta en initial_prompt de Whisper → mejora transcripción de esos términos
  • Se inyecta en system instruction de Gemini → fuerza ortografía correcta
  • El contexto guía cómo Gemini elige tools (sabe a qué te refieres con "Kanbox")

snippets.json — atajos de voz

{
  "snippets": {
    "firma mail": "Un abrazo,\\nCarlos Alcalá\\nPlaying Camp · playingcamp.com",
    "intro playing camp": "Hola, soy Carlos de Playing Camp..."
  }
}

Si dictas exactamente la clave (sin puntuación final, mayúsculas ignoradas) → pega el valor sin pasar por Gemini.

8. Troubleshooting

SíntomaCausa probableSolución
This process is not trusted! en err log Falta permiso de Accesibilidad / Input Monitoring para VoiceFlow Configuración → Privacidad → activar VoiceFlow en ambos
Tecla ⌥ no detectada Permisos no aplicados al binario actual tccutil reset Accessibility com.carlos.voiceflow y volver a conceder
"No tool call" o respuesta rara de Gemini Quota exhausted o conectividad Comprobar tail ~/voice-flow/voiceflow.err.log
Transcripción mala Modelo base demasiado pequeño Cambiar MODEL_NAME = "small" en voiceflow.py (más RAM y CPU)
Modo replace no captura selección App no responde a Cmd+C estándar Verificar manualmente que Cmd+C copia en esa app
Limitación TCC. Si recompilas VoiceFlow.app o le cambias el bundle ID, los permisos se invalidan. Hay que reañadirla manualmente o ejecutar tccutil reset Accessibility com.carlos.voiceflow.
